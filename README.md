## 🎮 О Игре

Plants vs Zombies Online — это многопользовательская браузерная игра, где вы можете сражаться с другими игроками в режиме реального времени. Выберите сторону: защищайте сад с помощью растений или атакуйте зомби!

### Особенности

- 🌐 **Мультиплеер** — играйте против реальных игроков
- 🤖 **Боты** — если нет противников, играйте с ИИ (3 уровня сложности)
- 📱 **Браузерная игра** — не требует установки, работает везде
- 💾 **Сохранение прогресса** — ваши монеты, уровень и достижения сохраняются

## 🎯 Основные Функции

### ⚔️ Боевая Система
- **5 линий** — размещайте юнитов на одной из пяти линий
- **Реальное время** — сражения происходят в реальном времени
- **Различные типы юнитов:**
  - 🌻 Атакующие — наносят урон противнику
  - ☀️ Солнцедобыватели — производят ресурсы
  - 🛡️ Защитники — имеют большое здоровье
  - 💥 Взрывные — наносят массовый урон
  - ✨ Особые — имеют уникальные способности

### 🏪 Магазин
- Покупка растений и зомби за монеты
- **Система уровней** — некоторые юниты доступны только с определённого уровня
- **Прокачка юнитов** — улучшайте юнитов до 10 уровня
- Скидки и акции от администрации

### 📈 Система Прогресса
- **Уровни игрока** — повышайте уровень, играя матчи
- **Опыт (XP)** — получайте опыт за победы
- **Монеты** — зарабатывайте за победы, квесты и достижения
- **Путь Славы** — уникальные награды за каждый уровень

### 🏆 Достижения
- 10 уникальных достижений
- Бонусы за выполнение
- Отслеживание прогресса

### 📋 Ежедневные Квесты
- 3 квеста в день
- Разнообразные задания
- Награды за выполнение

### 👥 Социальные Функции
- **Друзья** — добавляйте друзей, смотрите их профили
- **Лидерборд** — топ игроков по победам и уровню
- **История матчей** — просмотр последних игр

### ⚙️ Админ-Панель
- Создание промокодов
- Управление скидками
- Статистика сервера

## 🎮 Как Играть

1. **Зарегистрируйтесь** — создайте аккаунт
2. **Выберите режим:**
   - «Найти игру» — PvP против реального игрока
   - «Играть с ботом» — тренировка против ИИ
3. **Выберите сложность:** Легкий, Средний, Сложный
4. **Играйте:**
   - 🌻 Сторона растений: размещайте растения для защиты дома
   - 🧟 Сторона зомби: атакуйте, уничтожьте дом противника
5. **Зарабатывайте** — получайте монеты и опыт за победы

## 🛠️ Технические Детали

### Стек Технологий
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Node.js, Express
- **Real-time:** Socket.io
- **Database:** MongoDB (Atlas)
- **Hosting:** Render / Railway / VPS

### Требования
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Интернет-соединение
- Node.js 18+ (для локального запуска)

### Установка (Локально)

```bash
# Клонировать репозиторий
git clone https://github.com/gopp9745-oss/plants-vs-zombies-online.git

# Перейти в папку
cd plants-vs-zombies-online

# Установить зависимости
npm install

# Запустить сервер
npm start
```

Сервер запустится на http://localhost:3000
