# 🌻 Растения против Зомби - Онлайн

Мультиплеерная онлайн-игра "Растения против Зомби" с реальными игроками!

## 🌐 Игра

**Играйте онлайн:** https://plants-vs-zombies-online.onrender.com

## 🚀 Особенности

- 🎮 **Мультиплеер** - играйте против реальных игроков
- 🤖 **Игра с ботом** - 3 уровня сложности
- 🏆 **Рейтинги** - ELO система и сезоны
- 💰 **Магазин** - покупайте растения и зомби
- 📊 **Уровни** - прокачивайтесь и получайте награды
- 🎁 **Промокоды** - активируйте бонусы
- 👥 **Друзья** - добавляйте друзей
- ⚙️ **Админ-панель** - полное управление

## 🛠️ Установка локально

```bash
# Клонировать репозиторий
git clone https://github.com/gopp9745-oss/plants-vs-zombies-online.git
cd plants-vs-zombies-online

# Установить зависимости
npm install

# Запустить сервер
npm start
```

Откройте http://localhost:3000

## 🗄️ Требования

- Node.js 18+
- MongoDB (локально или в облаке)

## 🔧 Переменные окружения

```env
MONGO_URI=mongodb://localhost:27017
DB_NAME=pvz_online
PORT=3000
```
